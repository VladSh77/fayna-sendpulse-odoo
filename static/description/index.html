Odoo 17 Module

Fayna SendPulse Odo

Двостороння інтеграція SendPulse з Odoo — всі месенджери в одному вікні Discuss. Telegram, WhatsApp, Instagram, Facebook та інші без зміни інтерфейсу.

Можливості

Що вміє модуль

💬

Discuss інтеграція

Повідомлення з усіх месенджерів з'являються прямо в Odoo Discuss. Оператори відповідають — клієнт отримує у своєму каналі.

📋

Повна історія в картці

Вся переписка зберігається у вкладці Messaging картки партнера назавжди. Закриття розмови не видаляє листування.

🔍

Ідентифікація клієнтів

Автоматично за email або телефоном, або вручну через wizard. Нові контакти — в окремій черзі.

⚠️

Сповіщення про помилки

Якщо повідомлення не дійшло — оператор бачить причину в чаті: бот заблоковано, вікно 24h, відписка.

💡

Автовідповідь на коментарі

Публічна + приватна відповідь на коментарі під постами Facebook та Instagram через Graph API.

📊

UTM + CRM

Автоматична UTM-атрибуція для кожного каналу. Команда /lead — створює CRM-лід з чату.

Налаштування за 4 кроки

1
API ключі SendPulse

Вкажіть Client ID та Client Secret у Налаштуваннях → SendPulse Odo

2
Webhook у SendPulse

Додайте URL в кабінеті SendPulse для кожного бота: https://your-domain/sendpulse/webhook

3
Доступ операторів

Призначте співробітників до групи SendPulse Officer у Налаштуваннях → Користувачі

4
Facebook коментарі (опціонально)

Додайте Facebook Page Access Token для автовідповідей на коментарі FB / Instagram
