SendPulse Odo

Двостороння інтеграція SendPulse з Odoo — переписка з клієнтами прямо в Discuss

Функціонал

  • 💬 Discuss інтеграція — повідомлення від клієнтів з'являються прямо в Odoo Discuss
  • ↩️ Відповідь з Odoo — оператор відповідає у Discuss, клієнт отримує у своєму месенджері
  • 📋 Картка клієнта — повна історія переписки збережена у вкладці партнера
  • 🔍 Ідентифікація клієнтів — автоматично або вручну через wizard пошуку по email/телефону
  • 🔔 Нові чати зверху — непрочитані розмови завжди перші у списку
  • 📎 Вкладення — підтримка передачі фото та файлів
  • Telegram — відповідати можна будь-коли
  • Instagram — в межах 24-годинного вікна
  • Facebook / Messenger — в межах 24-годинного вікна
  • Viber, WhatsApp, TikTok — підтримка всіх каналів SendPulse
  • UTM атрибуція — автоматичне джерело трафіку для кожного каналу
  • Авто-синхронізація — cron кожні 10 хв підхоплює нові чати

Налаштування

1. Вкажіть Client ID та Client Secret від SendPulse API у Налаштуваннях → SendPulse
2. Додайте Webhook URL у кабінеті SendPulse для кожного бота: https://your-domain/sendpulse/webhook
3. Призначте операторів до групи SendPulse Officer у Налаштуваннях → Користувачі
