💬 SendPulse Odo

Двостороння інтеграція SendPulse з Odoo — вся переписка з месенджерів прямо в Discuss

Odoo 17 Community / Enterprise LGPL-3.0 v17.0.3.1.0

Функціонал

Робота з клієнтами

  • Discuss інтеграція — повідомлення з'являються прямо в Odoo Discuss
  • Відповідь з Odoo — клієнт отримує у своєму месенджері
  • Картка клієнта — повна історія переписки в партнері назавжди
  • Ідентифікація — автоматично або вручну по email / телефону
  • CRM /lead — команда в чаті створює ліда одним кліком
  • Вкладення — фото та файли в обох напрямках

Автоматизація та контроль

  • Нові чати зверху — непрочитані завжди перші у списку
  • Авто-синхронізація — cron кожні 10 хв підхоплює нові чати
  • Сповіщення про помилки — бот заблоковано, вікно 24h, відписка — оператор бачить причину в чаті
  • Збереження історії — закриття розмови не видаляє переписку з Discuss
  • UTM атрибуція — автоматичне джерело трафіку для кожного каналу
  • Автовідповідь на коментарі — публічна + приватна відповідь FB/Instagram

Підтримувані канали

✅ Telegram — будь-коли ⏱ Instagram — 24h вікно ⏱ Facebook / Messenger — 24h вікно ✅ WhatsApp ✅ Viber ✅ TikTok ✅ LiveChat

Налаштування

1
Вкажіть Client ID та Client Secret від SendPulse API у Налаштуваннях → SendPulse
2
Додайте Webhook URL у кабінеті SendPulse для кожного бота:
https://your-domain/sendpulse/webhook
3
Призначте операторів до групи SendPulse Officer у Налаштуваннях → Користувачі
4
(Опціонально) Додайте Facebook Page Access Token для автовідповідей на коментарі FB/IG
